## 1. 模块概述

### 1.1 功能简介
统计报表模块提供航空排班系统的各类统计报表和数据分析，包括：
- 机组排班统计报表
- 航班执行报表
- 飞行时长统计报表
- 规则校验报表
- 资质合规报表

支持报表生成、导出（Excel/PDF）、打印等功能，为管理决策和民航局审计提供数据支持。

### 1.2 用户角色
- **管理员 (admin)**：全部权限，可生成和导出所有报表
- **排班员 (roster)**：可生成和导出排班相关报表
- **机组人员 (crew)**：仅可查看个人相关报表
- **签派员 (oc)**：可查看航班和合规报表
- **监控员 (monitor)**：只读权限

### 1.3 数据来源
参考 `index.html` 中的"统计报表"部分

## 2. 功能清单

### 2.1 报表类型

| 报表名称 | 说明 |
|----------|------|
| 机组排班报表 | 11名飞行员的排班统计，含飞行次数、时长、执勤天数等 |
| 航班执行报表 | 2架货机的航班执行统计，含完成率、飞行时长等 |
| 飞行时长报表 | 飞行时长统计，含28天、14天、7天等周期 |
| 规则校验报表 | 规则校验统计，民航局检查专用 |
| 资质合规报表 | 机组资质合规报表，强制审计项 |

### 2.2 机组排班统计报表字段

| 字段 | 类型 | 说明 |
|------|------|------|
| 序号 | 数字 | 序号，从1开始 |
| 机组编号 | 文本 | 工号，如 C001 |
| 机组姓名 | 文本 | 姓名 |
| 职位 | 文本 | 机长/副驾/替补 |
| 飞行次数 | 数字 | 周期内飞行次数 |
| 飞行时长28d | 文本 | 28天累计飞行时长/上限，如 42h/190h |
| 飞行时长14d | 文本 | 14天累计飞行时长/上限，如 20h/95h |
| 飞行时长7d | 文本 | 7天累计飞行时长/上限，如 10h/55h |
| 执勤天数 | 数字 | 周期内执勤天数 |
| Standby次数 | 数字 | 周期内待命次数 |
| Standby时长 | 文本 | 周期内待命总时长 |
| DDO次数 | 数字 | 周期内强制休息次数 |
| 当前状态 | 文本 | 执勤/休息/待命 |
| 合规率 | 百分比 | 周期内合规百分比 |

### 2.3 航班执行统计报表字段

| 字段 | 类型 | 说明 |
|------|------|------|
| 序号 | 数字 | 序号，从1开始 |
| 机号 | 文本 | 飞机编号，如 AC1 |
| 机型 | 文本 | 机型，如 B747-F |
| 航班数 | 数字 | 周期内计划航班数 |
| 已执行 | 数字 | 已执行航班数 |
| 已完成 | 数字 | 已完成航班数 |
| 已取消 | 数字 | 已取消航班数 |
| 完成率 | 百分比 | 已完成/计划×100% |
| 执行率 | 百分比 | 已执行/计划×100% |
| 飞行总时长 | 时间 | 周期内累计飞行时长 |
| 平均时长 | 时间 | 单次平均飞行时长 |
| 合规率 | 百分比 | 合规航班/已完成×100% |
| 异常次数 | 数字 | 异常情况次数 |
| 状态 | 文本 | 正常/维修/停用 |

### 2.4 规则校验报表字段

| 字段 | 说明 |
|------|------|
| 序号 | 序号 |

### 2.5 资质合规报表字段

| 字段 | 说明 |
|------|------|

| 资质状态 | 合规/不合规 |
| 风险等级 | 安全/告警/危险 |

### 2.6 周期选项

## 6. 业务流程

### 6.1 生成报表流程

```
用户选择报表类型和周期
    ↓
点击"生成报表"
    ↓
系统查询相关业务数据
    ↓
按报表模板格式化数据
    ↓
渲染报表视图
    ↓
显示报表结果
```

### 6.2 导出报表流程

```
用户在报表查看界面点击"导出Excel/PDF"
    ↓
选择导出周期范围（可选，默认当前周期）
    ↓
系统生成报表文件
    ↓
返回文件下载链接
    ↓
用户下载文件
```

### 6.3 定时报表生成（可选）

```
系统定时任务触发（如每日凌晨）
    ↓
按配置的报表模板生成报表
    ↓
保存到报表历史
    ↓
（如需）发送报表到指定邮箱
```

## 7. 异常处理

| 场景 | 处理方式 |
|------|----------|
| 数据查询失败 | 提示"报表生成失败，请稍后重试" |
| 导出文件失败 | 提示"导出失败，请稍后重试"，记录错误日志 |
| 报表周期无数据 | 提示"当前周期无数据"，显示空白报表 |
| 导出格式不支持 | 提示"不支持该导出格式" |
| 周期内无排班数据 | 提示"当前周期无排班数据"，显示空白报表 |
| 某飞机无数据 | 显示"-"或0，不报错 |
