## 1. 模块概述

### 1.1 功能简介
登录认证模块是系统的基础安全模块，负责用户身份验证、会话管理、权限控制。包括：
- 用户登录/登出
- 账号注册（可选）
- 密码管理（修改、找回、重置）
- 会话管理（Token刷新、超时处理）
- 多因素认证（可选）
- 登录日志与安全审计

### 1.2 用户角色
| 角色 | 权限 |
|------|------|
| 管理员 (admin) | 系统全部功能 |
| 排班员 (roster) | 排班相关功能 |
| 机组人员 (crew) | 个人相关功能 |
| 签派员 (oc) | 航班相关功能 |
| 监控员 (monitor) | 只读监控功能 |

### 1.3 数据来源
参考系统设置模块中的用户表、角色表、权限表

## 2. 功能清单

### 2.1 登录功能

| 功能 | 说明 |
|------|------|
| 账号密码登录 | 用户名+密码登录 |
| 验证码 | 图形验证码/短信验证码（可选） |
| 记住登录 | 7天免登录 |
| 登录失败锁定 | 连续5次失败锁定30分钟 |
| 登录日志 | 记录登录时间、IP、设备 |

### 2.2 登出功能

| 功能 | 说明 |
|------|------|
| 主动登出 | 用户主动退出，清除Token |

| 超时登出 | Token过期自动登出 |

### 2.3 密码管理

| 功能 | 说明 |
|------|------|
| 修改密码 | 验证原密码后修改 |
| 找回密码 | 邮箱/短信验证后重置 |
| 密码强度校验 | 8位以上，包含大小写字母和数字 |
| 密码加密存储 | BCrypt强哈希 |

### 2.4 会话管理

| 功能 | 说明 |
|------|------|
| JWT Token | 生成包含用户信息的Token |
| Token刷新 | 主动刷新Token延长有效期 |
| 多端登录控制 | 支持单端/多端登录配置 |
| 会话列表 | 查看当前账号所有登录会话 |

### 2.5 安全功能

| 功能 | 说明 |
|------|------|
| 登录日志 | 记录每次登录的时间、IP、设备 |
| 操作日志 | 记录关键操作（新增、删除、审批等） |
| 异常登录告警 | 异地登录、频繁失败告警 |
| IP白名单 | 限制登录IP范围（可选） |

## 6. 业务流程

### 6.1 登录流程

```
用户输入账号密码
    ↓
系统校验账号是否存在
    ↓
账号不存在 → 返回"账号不存在"
    ↓
账号存在 → 校验账号状态（是否禁用/锁定）
    ↓
账号锁定中 → 校验是否已过锁定时间
    ↓
未过锁定时间 → 返回"账号已锁定，请X分钟后重试"
    ↓
校验验证码（如果开启）
    ↓
校验密码（BCrypt比对）
    ↓
密码错误 → 登录失败次数+1
    ↓
连续失败≥5次 → 锁定账号30分钟
    ↓
返回"密码错误，剩余X次机会"或"账号已锁定"
    ↓
密码正确 → 登录成功
    ↓
生成JWT Token
    ↓
记录登录日志
    ↓
返回Token和用户信息
```

### 6.2 Token刷新流程

```
前端检测Token即将过期（提前5分钟）
    ↓
调用刷新接口，携带refreshToken
    ↓
验证refreshToken有效性
    ↓
Token有效 → 生成新的Token对
    ↓
更新会话信息
    ↓
返回新的Token
    ↓
Token无效 → 返回401，前端跳转登录页
```

### 6.3 修改密码流程

```
用户输入原密码和新密码
    ↓
前端校验密码格式
    ↓
调用修改密码接口
    ↓
验证原密码是否正确
    ↓
原密码错误 → 返回"原密码错误"
    ↓
原密码正确 → 校验新密码不能与原密码相同
    ↓
新密码与原密码相同 → 返回"新密码不能与原密码相同"
    ↓
新密码加密（BCrypt）
    ↓
更新数据库密码
    ↓
生成新的Token（强制重新登录）
    ↓
记录操作日志
    ↓
返回修改成功
```

### 6.4 找回密码流程

```
用户输入账号
    ↓
系统校验账号是否存在
    ↓
账号不存在 → 返回"账号不存在"
    ↓
账号存在 → 发送验证码（短信/邮箱）
    ↓
验证码校验
    ↓
验证码错误 → 返回"验证码错误"
    ↓
验证码正确 → 进入密码重置页面
    ↓
输入新密码
    ↓
确认新密码
    ↓
密码加密（BCrypt）
    ↓
更新数据库密码
    ↓
记录操作日志
    ↓
返回重置成功，跳转登录页
```

## 7. 异常处理

| 场景 | 处理方式 |
|------|----------|
| 账号不存在 | 提示"账号不存在"，记录登录失败 |
| 密码错误 | 提示"密码错误，剩余X次机会"，记录登录失败 |
| 账号已锁定 | 提示"账号已锁定，请X分钟后重试" |
| 验证码错误 | 提示"验证码错误"，刷新验证码 |
| 验证码过期 | 提示"验证码已过期，请重新获取" |
| Token过期 | 返回401，前端跳转登录页 |
| Token无效 | 返回401，前端跳转登录页 |
| 修改密码原密码错误 | 提示"原密码错误" |
| 新密码不符合要求 | 提示"密码需8位以上，包含大小写字母和数字" |
| 两次密码输入不一致 | 提示"两次密码输入不一致" |
| 异地登录告警 | 发送告警通知（邮件/短信） |

## 9. 安全配置

### 9.1 密码策略

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 最小长度 | 8位 | 最少8位 |
| 必须包含大写字母 | 是 | 必须包含大写 |
| 必须包含小写字母 | 是 | 必须包含小写 |
| 必须包含数字 | 是 | 必须包含数字 |

### 9.2 登录安全策略

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 登录失败锁定次数 | 5次 | 连续失败5次后锁定 |
